# 🎯 타자 연습 게임 (Typing Practice Game)

C언어로 개발된 콘솔 기반 타자 연습 게임입니다. 다양한 난이도의 스테이지를 통해 타이핑 실력을 향상시킬 수 있습니다.

## 🎮 게임 특징

- **5단계 스테이지**: 점진적으로 어려워지는 난이도
- **실시간 타이핑 체크**: 입력과 동시에 정답 확인
- **시간 제한**: 스테이지별 시간 제한으로 긴장감 조성
- **랜덤 문제 선택**: 매번 다른 문제로 재미 유지
- **아름다운 UI**: ASCII 아트와 박스 형태의 깔끔한 인터페이스

## 📋 스테이지 구성

### 1단계: 기본 타이핑 (30초)

- 주어지는 문장을 정확히 입력
- 난이도: 매우 쉬움

### 2단계: 점 제거 타이핑 (30초)

- 점(.)을 제거하고 띄어쓰기 적용하여 입력
- 난이도: 쉬움

### 3단계: 자음/모음 복원 (20초)

- 일부 자음이나 모음이 지워진 문장을 완성
- 난이도: 보통

### 4단계: 특수문자 타이핑 (15초)

- 특수문자가 포함된 문장을 정확히 입력
- 난이도: 어려움

### 5단계: 단어 뒤집기 (15초)

- 각 단어를 개별적으로 뒤집어서 입력
- 난이도: 매우 어려움

## 🛠️ 개발 환경

- **언어**: C
- **컴파일러**: GCC
- **운영체제**: macOS, Linux
- **빌드 도구**: Make

## 📦 설치 및 실행

### 1. 저장소 클론

```bash
git clone https://github.com/your-username/typing-practice-game.git
cd typing-practice-game
```

### 2. 컴파일

```bash
make
```

### 3. 실행

```bash
./bin/typing_practice
```

## 🎯 게임 플레이 방법

1. 메인 메뉴에서 "1. 게임 시작" 선택
2. 각 스테이지의 규칙을 확인
3. Enter를 눌러 스테이지 시작
4. 제한시간 내에 목표 문장을 정확히 입력
5. 모든 스테이지를 클리어하면 게임 완료!

## 🔧 주요 기능

- **실시간 입력 처리**: 백스페이스 지원, 실시간 정답 체크
- **시간 관리**: 스테이지별 시간 제한 및 남은 시간 표시
- **문제 다양성**: Fisher-Yates 셔플 알고리즘으로 랜덤 문제 선택
- **UI/UX**: 깔끔한 박스 형태 인터페이스와 이모지 활용
- **에러 처리**: 입력 실패, 시간 초과 등 다양한 상황 처리
